# Relaxed Selection — Statistical Analysis

Source: `biodegen_perrun_all.csv`  |  task threshold = 5  |  eras: 0–50,000 (genes), 50,000–100,000 (+individual), 100,000–200,000 (+social)

All tests use the **run** as the unit of replication (one value per run). Each per-run value is averaged over a terminal 10,000-step window (≈25 snapshots) ending at the named timepoint, not a single tick, to remove instantaneous fluctuation. Effect sizes accompany every test because with dozens of runs even tiny differences reach significance — magnitude is the reviewers' real question.

Learning and social genes are not active in era 1 so their era-1 values reflect drift/genetic selection only.
